root help

Discuss is a networked electronic conferencing system, similar to the bulletin board programs found on many computer systems.

Please enter the hostname and the meeting name of the meeting you wish to access. A private meeting can be accessed if it has been set up for access via this gateway, by using the SSL mode with certificates.

Additional help is available on Web Certificates at MIT and Discuss on Athena.
