eTrac is the USC Employee Online Services website where all active USC Faculty, Staff, and student workers can access their employment information.
Just create a PIN and you log into the system.

If you have forgotten your PIN, simply set up a new one from the PIN Setup link located above. If you need help setting up a PIN, refer to the PIN Setup help.

eTrac loads in a popup window. If you do not see the eTrac login page or the PIN setup page, check the following:

  • The second browser window might have opened up behind this browser window. See if it's hidden behind this browser window.
  • Your browser is blocking popup windows, either by your browser configuration or a third-party popup window blocker. Disable the popup window browser, or re-configure the setting to allow popup windows from this site.

Your web browser must be Javascript enabled and allow popup windows.
Refer to the following AIS browser help pages for browser information and setup instructions:
  • Web Browsers FAQ
  • Recommended Browsers & Setup for Windows
  • Recommended Browsers & Setup for Macintosh

For more information about eTrac, see the eTrac Help Page.

If you need your access reset or have other questions about eTrac, contact your Home Department Coordinator or send email to eTrac@fismaster.usc.edu.
